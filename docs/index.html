Dynamic Diurnal Population Simulation

Agent-based modeling of day-to-night population distribution across urban environments using Monte Carlo methods to quantify uncertainty.

Explore Methodology View on GitHub

Simulation Insights

Urban systems analysis often requires dynamic time-of-day population insights rather than static residential counts. By leveraging Monte Carlo simulations, we explicitly model and quantify uncertainties in Work Start/End Times and Workplace Assignments.

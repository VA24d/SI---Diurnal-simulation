## Overview

Urban systems analysis often requires dynamic **time-of-day population** insights rather than static residential counts. This project provides a robust, simulation-based approach to estimate population distribution across buildings in Gachibowli over a typical 24-hour cycle. 

By leveraging **Monte Carlo simulations**, we explicitly model and quantify uncertainties in:
1. **Work Start/End Times**
2. **Workplace Assignments**
